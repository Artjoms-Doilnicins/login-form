# Login Form

[https://artjoms-doilnicins.github.io/login-form/](https://artjoms-doilnicins.github.io/login-form/)

This is a simple and stylish login form developed using HTML and CSS. The project demonstrates basic layout skills and provides a clean and functional user interface for a login page.

## Features

### HTML
- Input fields for username and password.
- Icons for each input field.
- A login button.
- Links for password recovery and user registration.
- A "Remember me" checkbox functionality.
- Modern styles using transparency, filters, and hover effects.

### CSS 

The login page uses modern design principles with a sleek, minimalist look and subtle hover effects. It is responsive and adapts well to different screen sizes.
 
- **Root Variables** : The page uses CSS root variables for easy customization of key properties, including font sizes, colors, and border radii. This ensures consistency and simplifies future adjustments.
 
- **Body Background** : The background is a full-screen image that gives the page a modern and visually appealing feel. It is fixed, ensuring it stays in place while the content scrolls.
 
- **Container** : The main form container has a subtle backdrop blur effect, combined with a transparent border to create a floating, soft look. It is centered on the screen using flexbox for easy alignment and responsiveness.
 
- **Input Fields** : The input fields for the username and password are styled with rounded corners and a soft border. The background is transparent to match the clean, modern look. Each input field is also paired with an icon for better visual identification.
 
- **Login Button** : The login button is styled with a soft white background and rounded edges. It changes opacity slightly when hovered to give a more interactive and polished feel.
 
- **Remember Me** : The checkbox for the "Remember me" option is neatly aligned with a small label next to it. The flex layout ensures that the elements are spaced evenly and appear clean.
 
- **Forgot Password & Register Links** : These links are styled with bold fonts, making them stand out. They are placed strategically to offer easy navigation without cluttering the design.
 
- **Responsive Design** : The layout is designed to be flexible and adjusts to different screen sizes. It ensures that the form remains user-friendly and visually appealing across devices, from desktop to mobile.
